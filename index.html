How to play ?

When it's your turn, use these commande :

  • Simple attack : yourName.dealDamage(player_you_want_to_attack)
  • Special attack : yourName.specialAttachkName(player_you_want_to_attack)

If you want to see status, use this :

game.watchStats()

If you want to change turn, use this :

game.startTurn()
